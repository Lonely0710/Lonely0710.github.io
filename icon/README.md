# Icon Directory

Keep icons grouped by purpose:

- `nav/`: navigation and route icons.
- `social/`: social, contact, and profile links.
- `actions/`: reusable action icons such as CV, link, or docs.
- `sections/`: section headers and card-type icons.
- `brands/`: product, platform, framework, language, and service logos.
- `misc/`: one-off assets such as 404 illustrations.

Prefer short, stable filenames. In code, use the helpers in
`components/ui/icon-paths.js`; in `IconStack`, either pass a helper-generated
absolute path or a categorized path such as `social/github.svg`.
